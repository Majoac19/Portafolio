Proyectos Destacados

Urban Scooter

Validé la calidad de una plataforma con aplicación web y móvil. El reto fue asegurar la consistencia entre API, base de datos y UI. Diseñé y ejecuté pruebas funcionales, de API en Postman y consultas SQL en Cygwin. Verifiqué flujos completos usando DevTools y Android Studio. Detecté y documenté discrepancias críticas entre frontend y backend, mejorando la estabilidad del sistema antes del release.

API Testing - SQL Pruebas en dispositivo móvil con Android Studio y página web con DevTools

Urban Lunch

Probé una app de seguimiento de pedidos donde el mayor desafío fue identificar fallos intermitentes en la comunicación con la API. Ejecuté casos de prueba en Android Studio, validé respuestas en Postman y analicé errores directamente en Logcat. Documenté bugs relacionados con validaciones, tiempos de respuesta y manejo de errores. Mis hallazgos ayudaron a mejorar la confiabilidad del seguimiento en tiempo real.

Logcat Postman

Urban Routes

Desarrollé un flujo de automatización con Selenium WebDriver para validar funciones clave de una app web. El reto fue cubrir escenarios críticos que fallaban de manera manual. Configuré el entorno, creé scripts en Python, apliqué waits, localizadores robustos y pequeñas aserciones. Automatizé el flujo principal reduciendo el tiempo de ejecución de pruebas recurrentes y complementando el proceso manual con validaciones más estables

Python Selenium DevTools
